# Archive

Deprecated, unused, or superseded content. Nothing in this directory is loaded at runtime.

## Quick Rules (from `markdown/archive-policy.md`)
- Move unused tools/scripts to `/archive/` with date prefix
- Every archived item must have an `INDEX.md` explaining why
- Delete if >180 days AND no references exist
- Never delete active game code — only tools, scripts, experimental content
