# Sound Library Add-On Batch GX-201 to GX-400

200 additional Suno Sound prompts focused on robot movement, hoverboard/flying, special tiles, digging feel, UI/level-up/rewards, discovery, combo feedback, scary depth presence, darkness, void, star formation, and galaxy sounds.

## Use

- Use the listed `Type` in Suno Sounds.
- Leave BPM and Key blank or Auto/Any.
- Save this file into `SoundLibrary_Review/00_PROMPT_SOURCE/`.
- Export Suno outputs into `00_INBOX_RAW_EXPORTS`, then sort using the review tags.

## Review Tags

`GOOD`, `GOOD_BUT_NEEDS_TRIM`, `GOOD_LAYER_ONLY`, `GOOD_ONE_SHOT`, `BAD_LOOP`, `TOO_MUSICAL`, `ROBOT_HUM`, `TOO_SYNTHETIC`, `HAS_VOICE`, `WRONG_SOURCE`, `KEEP_FOR_DEEP_LAYER`, `KEEP_FOR_VOID_LAYER`, `KEEP_FOR_UI`, `KEEP_FOR_OTHER_USE`
